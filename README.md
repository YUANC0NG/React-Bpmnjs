# React版本的Bpmnjs示例

### 自定义toolbar工具栏：
支持导入，导出，放大，缩小，前进，后退，重置，下载图片，

### 自定义属性面板:
支持设置审批角色
设置指定审批人
设置分支条件表达式
设置节点名称

### 快捷键功能
```
Undo	⌘ + Z
Redo	⌘ + ⇧ + Z
Select All	⌘ + A
Scrolling (Vertical)	⌥ + Scrolling
Scrolling (Horizontal)	⌥ + ⇧ + Scrolling
Direct Editing	E
Hand Tool	H
Lasso Tool	L
Space Tool	S
```
极简代码设计 帮助跟我一样有此需求的人学习理解
## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
